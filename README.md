# MyChat

**MyChat** — это десктопное приложение для обмена текстовыми сообщениями в реальном времени.

## ✨ Ключевые возможности

- Регистрация и авторизация через Firebase
- Создание приватных и групповых чатов
- Добавление контактов и поиск пользователей
- Обмен сообщениями в реальном времени через WebSocket
- Индикатор набора текста сообщения
- Кроссплатформенность: macOS, Windows, Linux

## 🏗 Архитектура

Клиент-серверная архитектура с WebSocket.
```text
                 ┌───────────────────────┐
                 │        Клиент         │
                 │      (Flutter)        │
                 └────────────┬──────────┘
          HTTP / WebSocket    │
      ┌───────────────────────┘
      ▼
┌─────────────────────────────┐
│         Сервер (Go)         │
│   бизнес-логика, валидация  │
└────────────────┬────────────┘
                 |
              Firebase
      ┌──────────┴──────────┐
      ▼                     ▼
┌────────────────┐    ┌──────────────┐
│  Auth          │    │  Firestore   │
│(аутентификация)│    │  (данные)    │
└────────────────┘    └──────────────┘
```

## 🛠 Технологический стек

| Технология                                      | Назначение                                      |
|-------------------------------------------------|-------------------------------------------------|
| Golang                                          | HTTP/WebSocket сервер (backend)                 |
| gorilla/websocket                               | Реальное время (WebSocket на сервере)           |
| Firebase Authentication + Firestore database    | Авторизация пользователей, хранение чатов и сообщений |
| Flutter (Dart)                                  | UI и логика клиента                             |
| web_socket_channel                              | Подключение к WebSocket-серверу (клиент)        |
| SharedPreferences                               | Хранение токенов авторизации на клиенте         |
## 🚀 Установка и запуск

### Способ 1: Запуск в режиме разработки

1. Клонируете репозиторий:
   ```bash
   git clone https://github.com/ivan-mezhenin/MyChat.git
   cd MyChat
   ```
2. Установите зависимости:
   ```bash
   cd frontend/my_chat
   flutter pub get
   ```
3. Запустите приложение:
   ```bash
   flutter run -d macos      # для macOS
   flutter run -d windows    # для Windows
   flutter run -d linux      # для Linux
   ```
### Способ 2: Готовый релиз

1. Перейдите в раздел Releases на GitHub:
https://github.com/ivan-mezhenin/MyChat/releases
2. Проверьте, что сервер доступен
3. Скачайте файл для своей платформы и следуйте инструкциям ниже:

#### macOS

- Скачайте: `MyChat.app.zip`
- Распакуйте архив и запустите приложение `MyChat.app`

**Важно!** macOS блокирует неподписанные приложения, поэтому нужно разрешить запуск вручную:

1. Перетащите `MyChat.app` в папку **Программы** (Applications).
2. Откройте **Системные настройки &rarr; Конфиденциальность и безопасность &rarr; Безопасность**.
3. Внизу появится сообщение «MyChat заблокирован». Нажмите **«Разрешить всё равно»**.
4. Откройте терминал:

   ```bash
   xattr -cr /path/to/MyChat.app
   ```
5. Если приложение всё равно не запускается, принудительно подписываем его:
   ```bash
   codesign --force --deep --sign - /path/to/MyChat.app
   ```
Готово! Приложение запустится без установки Flutter.
